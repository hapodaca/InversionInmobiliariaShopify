# Inversión Inmobiliaria Shopify Theme

Tema personalizado de Shopify para la tienda de Inversión Inmobiliaria.

## Estructura del Proyecto

```
├── assets/          # CSS, JavaScript, imágenes
├── config/          # Configuración del tema
├── layout/          # Layouts principales
├── locales/         # Traducciones
├── sections/        # Secciones reutilizables
├── snippets/        # Fragmentos de código
└── templates/       # Plantillas de páginas
```

## Desarrollo

Este tema está conectado con GitHub para control de versiones y con Shopify para deployment.

## Configuración

1. Clona este repositorio
2. Usa Shopify CLI para desarrollo local
3. Sube cambios a GitHub para mantener el historial
